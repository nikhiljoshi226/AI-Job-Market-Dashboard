## Project Overview

The AI Job Market Dashboard is a Power BI project created to analyze hiring and compensation trends in AI-related roles. The objective of the project is to present a structured view of the market by showing how salaries, skills, industries, and experience levels compare across different job titles.

The dashboard was designed to answer questions such as:
- Which AI job roles have the highest average salaries?
- How does salary change with experience level?
- Which technical skills appear most frequently across roles?
- How does skill demand vary across industries?

## Problem Statement
Job seekers, students, and professionals entering the AI field often find it difficult to understand which skills are most valuable, which roles pay more, and how industry expectations differ. This dashboard was built to organize that information in a visual and easy-to-understand format.

## Approach
- Collected and organized source datasets related to AI jobs and skill trends
- Cleaned and transformed the data in Power Query
- Modeled the data inside Power BI Desktop
- Built visuals to compare salary, skills, and market trends

## Output
The project resulted in an interactive dashboard that highlights:
- Average salary by job title
- Average salary by experience level
- Skill demand by role
- Skill distribution across industries

## Tools and Technologies
- Power BI Desktop
- Power Query
- CSV / Excel data sources
