# AI Job Market Dashboard

Power BI dashboard project analyzing AI job market trends, salary patterns, and skill demand across job roles, industries, and experience levels.

## Overview
This project was built in Power BI Desktop to explore how AI-related jobs differ by salary, experience level, job title, and skill requirements. The dashboard combines multi-source job market data and presents it through interactive visuals for easier comparison and insight generation.

## Key Features
- Average salary by job title
- Average salary by experience level
- Skill demand matrix across AI job roles
- Industry-wise skill mention analysis

## Tools Used
- Power BI Desktop
- Power Query
- CSV / Excel datasets
- Data cleaning and transformation
- Data visualization

## Dashboard Pages
### 1. Salary Insights by Job Title
Compares average salary across AI-related job roles.

### 2. Salary Insights by Experience Level
Shows salary differences across experience categories such as entry-level, mid-level, senior-level, and executive-level roles.

### 3. Skills Matrix by Role
Displays skill mentions across job titles to highlight which tools and technologies are most commonly associated with each role.

### 4. Industry and Skill Analysis
Shows how skill demand varies across industries.

## Data Preparation
Data cleaning and transformation were performed in Power Query within Power BI Desktop. The dashboard was built by shaping and modeling source datasets inside the `.pbix` file rather than exporting each intermediate cleaned table as separate CSV files.
